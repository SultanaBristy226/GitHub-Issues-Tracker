1. var, let, and const are used to create variables in JavaScript.
var works inside the function. let and const work inside the block {}.
const value cannot be changed after setting.

2. Spread operator (...) spreads or copies all items from an array or object.
It helps to copy or join arrays and objects easily.

3. map() changes each item and makes a new array.
filter() picks some items based on condition and makes a new array.
forEach() just runs a function for each item, but returns nothing.

4. Arrow function is a short way to write functions using =>.
It came in ES6 and is good for simple functions.

5. Template literals are strings written inside backticks (`).
We can put variables inside ${} and write multi-line strings easily.
